  Created a game derived from snake/ nimble quest, where a player character continuously moves until they either hit a wall, or die to the enemy npc. This code utilizes time.sleep to freeze the screen for 0.05 seconds in order to update the frame, in order to run sprite animation for each movement of the player or npc. Additionally recursive calls were used in order to check for hitbox/ enemy location, hp, and other similar character traits. Continuous movement was implemented using keyboard import.
